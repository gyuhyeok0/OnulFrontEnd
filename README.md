# 🎯 Onul Frontend (React Native)

모바일 피트니스 앱 **Onul**의 프론트엔드 레포지토리입니다.  
React Native 기반으로 iOS와 Android 모두 지원하며, 직관적인 UI와 부드러운 사용자 경험을 목표로 개발되었습니다.
